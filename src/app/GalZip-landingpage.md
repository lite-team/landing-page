### Design Requirements

**Visual Style**:
- Clean, minimalist design with plenty of whitespace
- Soft shadows and rounded corners (iOS-style)
- Smooth scroll-triggered animations (fade-in, slide-up)
- Parallax effects on hero section
- Micro-interactions on buttons (hover states, ripple effects)

**Typography**:
- San Francisco (SF Pro) font family (iOS native)
- Large, bold headlines (48-64px)
- Readable body text (16-18px)
- Clear hierarchy with proper spacing

**Responsive Design**:
- Mobile-first approach
- Optimize for iPhone screens (320px - 428px)
- Tablet/desktop versions with centered content (max-width: 1200px)
- Touch-friendly buttons (minimum 44pt tap targets)

**Animations**:
- Fade-in for sections as you scroll
- Slide-up for feature cards
- Smooth transitions between states
- Lottie animations for compression progress
- Parallax hero image

**Accessibility**:
- WCAG 2.1 AA compliance
- Proper heading hierarchy
- Alt text for all images
- Keyboard navigation support
- Sufficient color contrast (4.5:1 minimum)

### Technical Stack Suggestions
- **Framework**: React + Next.js or Vue + Nuxt (for SEO)
- **Animations**: Framer Motion or GSAP
- **Icons**: SF Symbols-inspired icon set
- **Hosting**: Vercel or Netlify
- **Analytics**: Privacy-friendly (Plausible or Fathom)
